## 评测范围

本评测默认不调用大模型生成最终答案，主要验证 RAG 证据链路是否可靠：

- 检索阶段是否命中期望来源
- prompt 引用上下文是否包含期望来源
- 命中文本是否覆盖关键事实
- 无依据问题是否被正确拒答

这样可以先稳定证明检索、引用和拒答逻辑的可复现性；最终回答质量可以在此基础上继续做人评或 LLM-as-judge。
